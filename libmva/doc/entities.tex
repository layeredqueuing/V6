\chapter{Entities}
\label{sec:entities}

\dfn{Entities} represent the software and hardware \dfn{nodes} in the
model.  Software entities are represented by
\link{\dfn{tasks}}{sec:task}.  These entities may also be used to
represent hardware devices such as disks.  Processors are modelled
specially in the model, hence are represented by their own class
\link{\file{Processor}}{sec:processor}.

Entities are used to group \link{\dfn{Entries}}{sec:entries} as the
entries are used to specify the behaviour in the overall layered
queueing network model.
